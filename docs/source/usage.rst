Usage
=====

.. _installation:

Installation
------------


To use CrevProp while in development, create a fork of the github repository.

.. code-block:: console

    from crevprop.iceblock import IceBlock


Initializing model
------------------

To initialize the model domian, use the ``IceBlock`` class:

.. code-block:: console

    geometry1 = IceBlock(1000,5,5,1,100)
